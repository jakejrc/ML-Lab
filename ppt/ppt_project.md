# ML-Lab PPT 项目定型

## 美学风格句
Deep Space Tech — 深空科技蓝，以深邃星空中极光光芒照亮数据科学之路。

## 调色板
| 角色 | 色值 | 用途 |
|------|------|------|
| 背景色 | #0a0e27 | 全局背景 |
| 强调色 | #00d4ff | 标题高亮、关键数据、装饰元素 |
| 金色点缀 | #ffd600 | 次级强调、数字高亮 |
| 主文字 | #ffffff | 标题、正文 |
| 辅助文字 | rgba(255,255,255,0.55) | 说明文字、注释 |

## 字体方案
- 展示字体：Arial（大标题，font-weight: 900，letter-spacing: 1px）
- 正文字体：Microsoft YaHei（正文，font-weight: 400）

## 设计思路
- 叙事结构：采用"问题发现 → 方案设计 → 功能展示 → 应用成效 → 创新亮点"的教学汇报弧线，每章以数据冲击页开场，正文页降低密度做功能拆解
- 节奏设计：封面强视觉冲击 → 问题页2页留白 → 方案页信息密度提升 → 功能展示页图表驱动 → 成效页大字数据收口 → 结尾简洁留白
- 视觉语言落地：深蓝背景贯穿全局，粒子光点装饰模拟星空；每页顶部极光线条作为视觉锚点；关键数字用金色#ffd600强调；功能截图/图表用浅色边框浮于深色背景上
- 场景感知：教育竞赛汇报场景，评委需要在8分钟内快速理解案例价值，每页聚焦一个核心信息，图多字少，功能页面优先用可视化截图代替文字描述

## 建议篇幅
约 16-18 页（案例汇报类PPT，信息密度适中，确保评委8分钟内完整理解）

## 逐页大纲

### #01 封面
- 类型：封面
- 构图：居中大标题 + 极光粒子背景
- 视觉焦点：ML-Lab 平台名称
- 风格关联：沿用方案1封面

### #02 目录
- 类型：导航页
- 构图：左侧4个章节编号 + 右侧简短描述
- 视觉焦点：章节编号的层次感
- 风格关联：极光竖线分隔左右

### #03 教学痛点（上）
- 类型：问题页
- 构图：2列布局，痛点图标+标题+一句话描述
- 视觉焦点：4个痛点卡片的视觉节奏
- 风格关联：红色警示色点缀痛点关键词

### #04 教学痛点（下）
- 类型：问题页
- 构图：数据冲击，大字体展示痛点量化
- 视觉焦点：关键数字（70%学生觉得ML抽象）
- 风格关联：金色数字强调

### #05 解决方案：ML-Lab
- 类型：方案页
- 构图：居中平台名称 + 一句话定位
- 视觉焦点：ML-Lab 平台Logo/名称
- 风格关联：极光背景 + 品牌色强调

### #06 平台架构
- 类型：架构图页
- 构图：4大模块水平排列 + 连线关系
- 视觉焦点：模块之间的数据流
- 风格关联：青蓝色模块卡片 + 白色连线

### #07 模块一：数据工作台
- 类型：功能展示页
- 构图：左文字说明 + 右功能截图区域
- 视觉焦点：数据分布可视化示意图
- 风格关联：浅色截图区浮于深蓝背景

### #08 模块二：算法实验室
- 类型：功能展示页
- 构图：6个算法图标/名称网格排列
- 视觉焦点：6大算法的可视化缩略图
- 风格关联：每个算法一个色块标识

### #09 算法可视化详解
- 类型：功能展示页
- 构图：3列布局，展示3个典型算法的可视化效果
- 视觉焦点：决策边界/决策树/损失曲线
- 风格关联：图表用浅色背景 + 彩色数据

### #10 模块三：模型评估台
- 类型：功能展示页
- 构图：左文字 + 右混淆矩阵/ROC曲线示意
- 视觉焦点：评估指标图表
- 风格关联：金色数字展示准确率

### #11 模块四：AI助教
- 类型：功能展示页
- 构图：对话界面模拟 + 功能说明
- 视觉焦点：AI对话交互效果
- 风格关联：聊天气泡样式模拟

### #12 AI赋能开发过程
- 类型：亮点页
- 构图：3步骤流程，展示AI辅助开发的Prompt→代码→成果
- 视觉焦点：通义千问辅助编程的过程截图示意
- 风格关联：流程箭头 + 金色步骤编号

### #13 应用成效
- 类型：数据页
- 构图：大字体数据 + 前后对比
- 视觉焦点：6大算法的准确率数据
- 风格关联：金色大数字 + 环形进度条

### #14 教学实践成果
- 类型：成果页
- 构图：学生竞赛获奖列表 + 课堂教学效果
- 视觉焦点：7项竞赛获奖
- 风格关联：奖杯图标 + 蓝色渐变

### #15 特色与创新
- 类型：总结页
- 构图：4个创新点水平排列
- 视觉焦点：4大亮点标题
- 风格关联：青蓝强调色 + 图标

### #16 致谢
- 类型：结尾页
- 构图：居中致谢文字 + 团队信息
- 视觉焦点：简洁留白 + 团队名称
- 风格关联：回归深空极光风格，与封面呼应
